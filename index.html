Hit 'enter' to start the mouse

Hit 'backspace' to stop the mouse



Currently the mouse is following a 'right hand rule.' If it senses that there is a right turn to be taken, it will take it first. However, if it senses that it's been on the path twice before, it will start prioritizing left hand turns. This rule causes some strange behavior. Depending on the maze, it can get stuck in an infinite loop, but also it might take right turns when the most optimal turn would have been left.
